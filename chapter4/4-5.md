### 4-5. 호출 표현식

<br/>
<br/>
- 함수나 메서드를 호출하는 문법
- 호출될 함수를 가리키는 함수표현식 + 괄호, 쉼표로 구성된 호출 인자의 목록

```js
f(0); //f는 함수 표현식, 0은 인자 표현식
Math.max(x, y, z); //Math.max는 함수, x,y,z는 호출 인자
```

<br/>
<br/>

- 호출 표현식이 평가될 때 순서

1. 함수표현식 평가(함수표현식의 값이 호출할 수 없는 객체일 경우 TypeError 발생)
2. 호출인자 표현식이 순서대로 평가
3. 함수 선언 시 지정했던 인자 이름에 인자 값이 차례로 할당
4. 함수 본문 실행
